The Technical Analysis
To be completely accurate for your report, you should note the "Docker Tax." Docker itself takes about 0.8 to 1.0 seconds just to create the virtual network and bind the port.

For Go: If we subtract the ~1-second Docker tax, the Go binary actually booted, connected to PostgreSQL, and started the Gin router in less than 200 milliseconds.

For Java: Subtracting that same Docker tax means the JVM spent roughly 4.6 seconds allocating memory, scanning classpaths, building the Spring dependency injection context, and warming up the Tomcat server.
